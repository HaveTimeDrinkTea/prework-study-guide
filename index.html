HTML

  • The head element contains inforamtion about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: check what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • The Null and Undefined Data types: Undefined denotes absence of a value and Null denotes absence of an object. See JavaScript Data Types
  • Note the special number type NaN. It is the only javascript value that is not equal to itself.
  • An object is a value in memory which is possibly referenced by an identifier. Objects can be seen as a collection of property-value pairs.
  • Functions are objects which has the additional capacbility of bein callable.
  • Read up on Type Coercion!
